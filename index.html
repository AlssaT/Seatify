What this actually is

This is a demo I made to keep track of course seats in a simple way. It does not connect to any real university system. It is just a front end simulation using fake data.

In a real version, there would be a backend checking the actual portal and sending real notifications. Here I just change the data in JavaScript on a timer.

Demo only. No real student data, no real university connection. Made for my portfolio.
